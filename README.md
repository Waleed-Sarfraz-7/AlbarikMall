# AlBarik Mall — Shopping Mall Management System

A Windows Forms desktop application built in C# for managing a shopping mall. Developed as a combined OOP and Database project in 2nd semester at UET Lahore.

---

## Features

**Multi-role access** — four user types with separate interfaces and permissions:
- **Admin** — full system control, manage users, shops, and employees
- **Manager** — product and employee management, profit/loss analysis, announcements
- **Ambassador** — billing, transaction processing, feedback collection
- **Customer** — browse products, view transactions, manage profile

**Modules**
- Employee and product management
- Billing and transaction history
- Shop management
- Profit/loss analysis
- Announcements
- Feedback collection
- Sign in / sign up with profile and password management

---

## Tech Stack

- C# / .NET Framework 4.7.2
- Windows Forms (WinForms)
- SQL database with a layered architecture (UI → Data Access Layer)

---

## Getting Started

**Requirements**
- Windows OS
- Visual Studio 2019 or later
- .NET Framework 4.7.2

**Run**
1. Clone the repo
2. Open `Final_Project` in Visual Studio
3. Set up the database connection string in the data layer
4. Build and run
